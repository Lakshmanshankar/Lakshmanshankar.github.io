This Repository contains the source code for my portfolio website. It is built using the [NextJS](https://nextjs.org/) and [TailwindCSS](https://tailwindcss.com/) frameworks.

## Getting Started

To get started, clone the repository and install the dependencies:

```bash
git clone git@github.com:Lakshmanshankar/Lakshmanshankar.github.io.git
cd Lakshmanshankar.github.io

# Install dependencies
npm install
```

## Development

To start the development server, run the following command:

```bash
npm run dev
```

After you have started the development server, you can view the website at [http://localhost:3000](http://localhost:3000).

Make changes to the code.
